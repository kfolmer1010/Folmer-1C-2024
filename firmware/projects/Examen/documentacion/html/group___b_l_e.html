#define BLE_NO_INT   0

BLE driver for the ESP-EDU Board.

Nota
This driver emulates HM-10 functionalities (same services and characteristics), so it can be used to communicate with common Android apps, like "Bluetooth Electronics" (https://play.google.com/store/apps/details?id=com.keuwl.arduinobluetooth)
